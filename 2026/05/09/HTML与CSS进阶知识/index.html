一、高级布局（Flexbox / Grid）

1. Flexbox

Flexbox 本质上是：

一维布局系统。

适合：

  • 一行布局
  • 一列布局
  • 对齐
  • 空间分配

display:flex

开启 flex 布局后：

  • 子元素变成 flex item
  • 默认沿主轴排列

主轴与交叉轴

  • row：主轴水平
  • column：主轴垂直

justify-content

控制主轴对齐。

align-items

控制交叉轴对齐。

flex-grow

控制剩余空间分配。

flex-shrink

控制压缩规则。

禁止压缩。

flex-basis

定义基础宽度。

圣杯布局

2. Grid

Grid 本质：

二维布局系统。

相比 Flex：

  • Flex 更适合线性布局
  • Grid 更适合复杂页面布局

例如：

  • 后台系统
  • 仪表盘
  • 卡片系统

创建 Grid
